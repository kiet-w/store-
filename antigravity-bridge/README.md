# Antigravity GPT & Claude Bridge

Secure bridge server to connect both **ChatGPT Web (via Custom GPT Actions)** and **Claude Web (via Remote MCP Connectors)** with your local Google Antigravity Agent.

## Architecture

```text
  [ChatGPT Web (Custom GPT)]             [Claude Web (Custom Connector)]
             ↓                                         ↓
Action (OpenAPI POST request)           Remote MCP (SSE Connection & POSTs)
             ↓                                         ↓
   ┌────────────────────────────────────────────────────────┐
   │             Tunnel (ngrok / Cloudflare)                │
   └────────────────────────────────────────────────────────┘
                               ↓
                   Local API Server (FastAPI)
                               ↓
     ┌──────────────────────────────────────────────────┐
     │           Antigravity Python Agent               │
     │      (Direct filesystem / shell tools)           │
     └──────────────────────────────────────────────────┘
```

## Setup & Running

### 1. Prerequisites
Make sure dependencies are installed inside the virtual environment:
```bash
cd ~/Projects/project/antigravity-bridge
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Start the Bridge Server
Run the FastAPI application locally:
```bash
python -m uvicorn bridge_server:app --host 127.0.0.1 --port 8000
```

### 3. Retrieve Your API Key
Open `~/Projects/project/antigravity-bridge/config.json` to view your secure autogenerated key (e.g., `100a1059-2c3c-492c-9806-425b2a9edb94`).

## Connecting Claude Web (Custom Connectors Beta)

Claude Web now supports adding remote MCP servers directly from the browser interface!

1. Start your tunnel (e.g. ngrok):
   ```bash
   ngrok http 8000
   ```
2. Copy your public forwarding URL (e.g., `https://xxxx.ngrok-free.dev`).
3. Open Claude Web → **Settings** (or click on your profile) → **Add Custom Connector** (or look for the Beta connectors section).
4. Fill in the form:
   * **Name:** `Antigravity`
   * **Remote MCP server URL:** `YOUR_TUNNEL_URL/mcp/sse?token=YOUR_API_KEY`
     *(Example: `https://tutor-egomaniac-tasty.ngrok-free.dev/mcp/sse?token=100a1059-2c3c-492c-9806-425b2a9edb94`)*
5. Click **Add**. Claude Web will establish a secure connection and automatically register the following tools:
   * `list_directory`: Quick local directory scanner.
   * `view_file`: Quick file contents viewer.
   * `write_file`: Local file editing tool.
   * `run_command`: Local shell command executor.
   * `ask_antigravity`: Multi-step reasoning tool using the full Google Antigravity Agent.

## Connecting ChatGPT Web (Custom GPT Actions)

1. Go to [ChatGPT Web](https://chatgpt.com) → **Explore GPTs** → **Create a GPT** → **Configure** → **Create new action**.
2. Set **Authentication** to **API Key** -> **Bearer** and paste your `api_key`.
3. Paste the OpenAPI JSON Schema from the `README.md` file (remembering to replace the server URL with your ngrok address).
