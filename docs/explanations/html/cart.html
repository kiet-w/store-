Module Cart (Giỏ hàng) - Tài liệu kỹ thuật

Quản lý vòng đời giỏ hàng của khách hàng, tích hợp Redis cache và tính toán tiền tệ chính xác.

Module Name: CartModule
Base Route: /cart
Files: cart.controller.ts, cart.service.ts
Dependencies: PrismaService, RedisService, AuthModule
Core Responsibility: CRUD giỏ hàng, stock validation, cache management.

Sơ đồ Hệ Thống (System Data Flow)

Phương thức

Controller: getCart
Controller: addItem
Controller: updateItem
Service: getCart
Service: addItem
Service: validateProductAvailability
Service: serializeCart

getCart()

GET /cart

Lấy thông tin giỏ hàng của user hiện tại từ Service.

graph LR
A[Client] -->|JWT| B(CartController)
B --> C{"CartService.getCart"}
C --> D[ApiResponse]

addItem()

POST /cart/items

Thêm sản phẩm vào giỏ hàng thông qua DTO.

graph LR
A["Body: AddCartItemDto"] --> B(Controller)
B --> C["Service.addItem"]
C --> D["Refresh Cache"]

Service: getCart(userId)

Internal Service

Sử dụng chiến lược Cache-aside để tối ưu hiệu năng.

graph TD
A[Start] --> B{Check Redis}
B -- Hit --> C[Return Data]
B -- Miss --> D[DB Query]
D --> E[Serialize]
E --> F[Set Redis]
F --> C

Service: addItem(userId, dto)

Internal Service

Thêm sản phẩm kèm kiểm tra stock và gộp số lượng.

graph TD
A[Start] --> B[Validate Stock]
B --> C[Get Active Cart]
C --> D{Existing Item?}
D -- Yes --> E[Update Qty]
D -- No --> F[Create Item]
E --> G[Refresh Cache]
F --> G

validateProductAvailability()

Private Helper

Kiểm tra tập trung: Tồn tại, Trạng thái ACTIVE, Soft-delete và Tồn kho.

graph LR
A["Check Product"] --> B{Active?}
B -- No --> C[Error]
B -- Yes --> D{"Stock >= Qty?"}
D -- No --> C
D -- Yes --> E[Success]

Private Helper

Chuyển đổi dữ liệu Decimal thành String và tính toán tổng tiền.

graph LR
A["Raw DB Data"] --> B["Calc Subtotal"]
B --> C["Decimal to String"]
