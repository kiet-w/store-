Hệ thống Prisma ORM

Lớp truy cập dữ liệu trung tâm, cấu hình cho môi trường Enterprise với PostgreSQL và Driver Adapters.

THÔNG TIN MODULE

File Paths:
  • prisma/schema.prisma
  • src/prisma/prisma.module.ts
  • src/prisma/prisma.service.ts
Core Responsibility: Quản lý connection pool, thực thi query, và đảm bảo Type Safety cho database.

Sơ đồ Hệ Thống (Data Access Flow)

Chi tiết Service

Class Name: PrismaService Constructor Logic: Khởi tạo pg.Pool và sử dụng PrismaPg adapter để tối ưu hóa việc quản lý kết nối. Visual Logic: 
graph TD
Start[Constructor] --> Env[Get DATABASE_URL]
Env --> Pool["Create pg.Pool"]
Pool --> Adapter["Create PrismaPg Adapter"]
Adapter --> Super["super({adapter})"]
Method Name: onModuleInit() Purpose: Đảm bảo kết nối database được thiết lập ngay khi ứng dụng khởi động. 
graph LR
Init["NestJS Init"] --> Conn["$connect"]
Conn --> Success["App Ready"]
Conn --> Fail["App Crash on Start"]

Chi tiết Schema (Cấu trúc dữ liệu)

Preview Features: ["driverAdapters"] Indexing Strategy: @@index([userId, status]) trên Cart và Order để tối ưu hóa các truy vấn lọc theo trạng thái của người dùng. 
graph TD
Schema["schema.prisma"] --> Models["User, Product, Cart, Order, etc."]
Models --> Enums["UserRole, ProductStatus, etc."]
Models --> Relations["One-to-Many, Many-to-One"]
